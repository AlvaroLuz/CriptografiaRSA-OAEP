Programa que simula o funcionamento de um algoritmo de criptografia RSA tipo oaep.

Feito para realizar Encriptação e Decriptação de mensagens. Ao encriptar-se uma mensagem, pode-se gerar uma chave nova a cada execução. A chave gerada e a mensagem encriptada se encontrarão no diretório em que o programa se encontra.

Instruções de uso constam na interface do programa.

Feito em python 3.8.12 com bibliotecas nativas da linguagem.

Comando de execução:
>python3 main.py
